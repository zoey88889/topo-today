第九课｜如何验证合约并部署到主网

在上一课中，我们了解了如何创建一个最简单的 ERC-20 Token 合约。本节课我们将深入学习如何将合约部署到以太坊主网，并进行验证，让你的 Token 对所有人可见、可信。

🛠 准备工作

  • 安装浏览器钱包 MetaMask
  • 在 Remix IDE 打开你的合约
  • 确保你拥有足够的 ETH 用于部署
  • 连接 Remix 与 MetaMask 的 Injected Provider

🚀 部署步骤

  1. 打开 Remix IDE，加载合约
  2. 选择 "Injected Provider - MetaMask"
  3. 编译合约后，点击 Deploy 按钮
  4. MetaMask 弹出交易签名窗口，确认并等待交易完成

🔍 验证合约

部署完成后，拿到你的合约地址，前往：

  • Etherscan（主网）
  • Sepolia 测试网

找到你的合约页面后，点击 Contract → Verify and Publish

✅ 提交验证信息

  • 选择 Compiler 版本与你在 Remix 编译时一致
  • 输入 License 类型（如 MIT）
  • 将你的合约代码粘贴进去，点击 “Verify”
  • 等待几秒后，合约源码将公开显示在区块浏览器上
提示：
Etherscan 会自动识别你是否使用了构造函数参数，请根据需要填写。

🎉 完成部署与验证

现在你就拥有了一个部署在区块链上的公开可查的 Token 合约，用户可以通过钱包添加你的 Token 地址，开始交易、转账。
